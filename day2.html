HTML Tables Introduction

Tables are used to display tabular data in rows and columns.

Basic tags:

  • <table>, <tr> (table row)
  • <td> (table data cell)
  • <th> (table header cell)
Slide 2 of 30

Rowspan and Colspan Attributes

Allow cells to span multiple rows or columns respectively:

  • rowspan="2" merges cell vertically in 2 rows
  • colspan="3" merges cell horizontally in 3 columns

Example:

      <td rowspan="2">Merged Vertical Cell</td>
      <td colspan="3">Merged Horizontal Cells</td>

HTML Forms Overview

Forms allow users to input data which can be sent to a server or processed.

  • <form> container for form elements
  • Input elements: <input>, <textarea>, <button>
Slide 5 of 30

Input Types in Forms

Common input types include:

  • text - single-line text input
  • password - masked password input
  • email - email address input
  • radio - select one option from many
  • checkbox - toggle options
  • date - date picker
Slide 6 of 30

Textarea and Button Elements

<textarea> is a multi-line text input:

      <textarea name="comments" rows="4" cols="50"></textarea>

<button> element triggers form actions

Slide 7 of 30

Labels and Accessibility

<label> associates text with form fields for better usability and accessibility.

Example:

      <label for="email">Email:</label>
      <input type="email" id="email" name="email">
Slide 8 of 30

Fieldset and Legend

<fieldset> groups related form elements.

<legend> gives a caption to the group.

      <fieldset>
        <legend>Personal Info</legend>
        <label>Name:</label> <input type="text">
      </fieldset>

Embedding Media in HTML

HTML5 supports embedding audio, video, and external content through:

  • <audio> tag for sound files
  • <video> tag for video files
  • <iframe> tag for embedding external content (e.g. YouTube)
Slide 11 of 30

Audio Tag

Supports multiple audio formats with fallback.

      <audio controls>
        <source src="audio-file.mp3" type="audio/mpeg">
        Your browser does not support the audio element.
      </audio>
Slide 12 of 30

Video Tag

Embed videos with playback controls.

      <video width="320" height="240" controls>
        <source src="video.mp4" type="video/mp4">
        Your browser does not support the video tag.
      </video>
Slide 13 of 30

Iframe Tag for Embedding YouTube Videos

Integrates external web content like videos:

      <iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
      frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" 
      allowfullscreen></iframe>

Best Practices for Forms

  • Use semantic HTML elements
  • Label every input clearly
  • Group related inputs with fieldset and legend
  • Validate required fields
  • Use placeholders carefully
Slide 18 of 30

Security in Forms

  • Never trust user input; validate on server
  • Use HTTPS for secure data transmission
  • Avoid exposing sensitive info in URLs
  • Implement CAPTCHA to avoid bots

Responsive Tables and Forms

  • Prefer flexible widths and stacking for mobile
  • Avoid fixed widths for input fields and tables
  • Use CSS media queries to adjust layout on small screens
Slide 23 of 30

Form Validation

  • Use HTML5 validation attributes: required, pattern, type
  • Use custom validation with JavaScript for better UX
  • Always validate on the server side for security
Slide 24 of 30

Multimedia Formats Support

  • Audio: mp3, ogg, wav
  • Video: mp4, webm, ogg
  • Browser compatibility varies; provide multiple sources if possible
Slide 25 of 30

Embedding Content Responsively

Make iframes and videos responsive by wrapping in a container with CSS.

      .responsive-iframe {
        position: relative;
        padding-bottom: 56.25%;
        height: 0;
        overflow: hidden;
      }
      
      .responsive-iframe iframe {
        position: absolute;
        top: 0; left: 0;
        width: 100%;
        height: 100%;
        border: 0;
      }

  • Maintain semantic structure
